# Changes Made to Project Plan

After meeting with the TA’s, there were a couple of adjustments we had to make. We also made changes after re-reading the submission requirements for the final project submission

  

Changes made:

-   Updated the datasets so that they are all in csv format. We previously mentioned using scraping for data, but ultimately decided that the csv’s we were able to find online were stronger and easier to integrate
    
-   Converted all files that were in the repository to markdown to make grading easier for TAs
    
-   Added new tasks to properly reflect the submissions that we need to make for our final project submission.
    
-   One additional change we made is that we are changing the timeframe of the data from the last 20 years to the entire history of the team. The reason we did this is that we decided the more historical data, the better, and we can use averages to determine which city is truly the best, rather than which city’s sports teams have been around the longest
    
-   The biggest change that we made was to one of the datasets. Our old mlb dataset was extremely hard to clean and match up to the others for analysis. We decided to go with a new MLB dataset that is sourced from a website that is very similar to the NFL and NBA datasets we used. This made cleaning and eventually merging much easier
    

  
  

# Challenges/Problems Incurred

-   One problem that we incurred was dealing with the uploading of our datasets into OpenRefine, but after trial and error, we were able to make it work and clean the individual datasets
    
-   Another challenge related to integration was determining what we wanted our naming system to be so that each city was easy to find and the combined dataset was easy to follow. Our integration is not yet complete but all of the cleaned files share the same columns and once integration is complete we will be ready for analysis
    
-   There are many challenges relating to the data quality of the NBA as there were many different versions of the league before it officially became the NBA. There are teams that are associated with the previous leagues like BBA and ABA, so we needed to do a lot of work when it came to cleaning
    
-   Another issue is that teams actually move cities. This is not as common in the last couple of decades, but since we are using data that dates back decades, we had to come to a decision as to how we were gonna treat these instances of teams that have moved cities.
